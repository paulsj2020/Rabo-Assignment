Follow the steps to run the application

1. Use "npm install" command. 
2. once the node modules is installed completely, go to package.json. 
3. Use "ng serve" to start the application. 
4. If you wish to host the application in your IP, edit the Script object in package.json like this "dev" : "ng serve --host=<your-ip-address> --port=<portno> --disableHostCheck". 

Note : Filer is applied for the field "IssueCount" only
